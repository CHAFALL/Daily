아래에 같은 것 복사 붙여넣기 : Ctrl + D
테스트 만들기: Ctrl + Shift + T
변수 뽑아오기: Ctrl + Alt + V
원하는 곳으로 옮기기: Alt + Shift +방향키
생성자 생성 : Alt + Insert
파일 내에 찾기: Ctrl + Shift + F
합치기(인라인): Ctrl + Alt + N
인자 꺼내주기: Ctrl + Alt + P
리컴파일: Ctrl + Shift + F9
최근에 쓴 리소스 보기: Ctrl + E
컬럼 선택 모드: Ctrl 연속 2번 누르고 2번째는 누른채로 방향기로 커서를 조절, 그 상태에서 Ctrl 쓰면 스마트하게 이동함
